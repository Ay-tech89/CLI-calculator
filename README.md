# 🧮 CLI Calculator

A command-line calculator built with Python.  
It supports full expression evaluation, a guided step-by-step mode, and safe execution using AST to prevent code injection.

---

## Features

-  Evaluate full mathematical expressions  
  Example: `(10 + 5) * 2`

-  Safe evaluation using Python's `ast` module

-  High-precision arithmetic using `decimal.Decimal`

- Supported operations:
  - Addition `+`
  - Subtraction `-`
  - Multiplication `*`
  - Division `/`
  - Power `^` or `**`
  - Modulus `%`
  - Square root `sqrt(x)`

- Built-in constants:
  - `pi`
  - `e`

- Step-by-step calculation mode

- Colored CLI output

- Clear screen support
